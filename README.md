# Object Detection and Segmentation Dataset Annotation

## Overview
This project demonstrates dataset annotation for object detection tasks. It includes **part-based annotations** using **bounding boxes** and **segmentation masks**, prepared using Roboflow.

## Tools Used
- Roboflow
- COCO dataset format

## What’s Included
- **Annotations/** – sample COCO annotation files (bounding box and segmentation)  
- **Annotation Screenshots/** – visual examples of annotated images and Roboflow interface

## Notes
Only **sample annotations** are included in this repository. Full dataset images are not provided due to project restrictions. The annotations follow COCO format standards and demonstrate structured labeling for object detection and segmentation tasks.
